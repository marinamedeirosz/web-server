Sobre o Projeto

Início Sobre Equipe

Este projeto é um servidor web multithread simples desenvolvido em Java para a disciplina de Fundamentos de Redes de Computadores.

Ele implementa um servidor TCP que atende requisições HTTP, servindo arquivos estáticos da pasta wwwroot.

O código foi baseado no desafio Coding Challenges - WebServer.

O objetivo é demonstrar conceitos de sockets, multithreading, protocolos HTTP e segurança básica contra ataques como path traversal.

Projeto criado para a disciplina de Fundamentos de Redes de Computadores.

Repositório no GitHub: https://github.com/marinamedeirosz/web-server
