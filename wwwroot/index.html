Servidor Web Multithread em Java

Início Sobre Equipe

Descrição do Projeto

Este projeto implementa um servidor web simples em Java que escuta conexões TCP, processa requisições HTTP e serve arquivos estáticos de forma multithread, permitindo atender vários clientes simultaneamente.

Como Usar

  1. Coloque seus arquivos HTML e TXT dentro da pasta wwwroot.
  2. Compile os arquivos Java na pasta src (separados) ou diretamente no diretório do projeto.
  3. Execute o servidor usando: java MultiThreadedWebServer [porta] (padrão 8080).
  4. Acesse no navegador: http://localhost:8080.

Tecnologias Utilizadas

  • Java SE (Sockets, Threads)
  • HTTP 1.0 - Protocolo para comunicação cliente-servidor
  • HTML e CSS para páginas estáticas

Projeto criado para a disciplina de Fundamentos de Redes de Computadores.
